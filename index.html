Abstract Machines & Analysis Tools

A collection of compiler phases, abstract machines, and tools based on abstract interpretation.

  • analyses Haskell
    Abstract interpretation analyses (Interval analysis, etc.).
  • cek Haskell
    CEK machine implementation.
  • cesk Haskell
    CESK machine implementation.
  • cesk-anf Haskell
    CESK machine implementation for A-Normal Form.
  • cesk-star Racket
    CESK* machine implementation.
  • cesk-star-timestamp Racket
    CESK* machine implementation with timestamps.
  • cls OCaml
    CLS machine implementation.
  • cls_draft OCaml
    Draft CLS machine implementation.
  • cls_store OCaml
    CLS machine implementation with store.
  • concrete
    Concrete interpreter tasks (Lambda calculus, etc.).
  • absint-tutor
    Abstract Interpretation Tutor / Notes.
  • use-redex-dsl Racket
    Racket Redex models for abstract machines.
